## Phase 1 — HR Analytics

**Dataset:** 1,470 employees · 38 columns
**Tools:** SQL Server (SSMS) · Power BI

| Layer | Topic | Status |
|---|---|---|
| Cleaning | Data prep & validation | ✅ Done |
| Layer 1 | Workforce Composition | ✅ Done |
| Layer 2 | Attrition & Retention | ✅ Done |
| Layer 3 | Compensation Analysis | ✅ Done |
| Layer 4 | Satisfaction vs Attrition | 🔄 Week 2 |
| Layer 5 | Performance & Overtime | 🔄 Week 2 |
| Dashboard | Power BI | 🔄 Week 2 |

**Week 1 Key Findings:**
- R&D dominates headcount at 67% — product-driven org profile
- 1 in 6 workers is a contractor (16.6%)
- Contractor Doctors earn 64% more than Full-time equivalents
- Pay appears role-based not education-based (Masters earn less than Bachelors)
